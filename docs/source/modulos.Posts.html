Vista de inicio “home_view”.

Esta vista verifica si el usuario está autenticado y obtiene todos sus permisos. Dependiendo de los permisos del usuario, agrega nombres específicos a la lista “sitios” para determinar qué secciones o funcionalidades se deben mostrar en la página de inicio.

req (HttpRequest): El objeto de solicitud HTTP.

Returns:

HttpResponse: La respuesta HTTP con el contenido renderizado de la plantilla “pages/home.html”.

Vista de detalle de publicación “PostDetailView”.

Esta vista muestra los detalles de un solo objeto “Post”. Utiliza el modelo “Post” para recuperar la instancia específica que se va a mostrar y utiliza la plantilla “posts/post_detail.html” para renderizar el contenido.
